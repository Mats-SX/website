What is openCypher?

openCypher is an open source project to broaden access to the industry’s most widely adopted graph query language: Cypher.

Focus on Your Domain

Graphs naturally describe your domain, and Cypher lets you focus on that domain instead of getting lost in the mechanics of data access. Both expressive and efficient, Cypher eliminates the distraction of learning a query language.

Human Readable

Cypher is a human readable query language that makes complex things possible. A combination of English prose and neat iconography means Cypher is accessible to both developers and operations professionals.

Complete Open Source Access

The openCypher project means you can use Cypher as your query language for graph processing capabilities within any product or application. Everyone from Oracle and SAP to Microsoft and IBM can (and does) use Cypher’s capabilities – now you can too.

                        
        MATCH (cypher:QueryLanguage)-[:QUERIES]->(graphs)
        MATCH (cypher)<-[:USES]-(u:User) WHERE u.name IN [‘IBM’, ‘Oracle’, ‘SAP’, ‘Microsoft’]
        MATCH (openCypher)-[:MAKES_AVAILBLE]->(cypher)
        RETURN cypher.attributes
