openCypher Project

openCypher is an open source project to bring a new public implementation of the industryâ€™s most widely adopted graph query language: Cypher.

The openCypher project aims to improve growth and adoption of graph processing and analysis by providing an open graph query language to any data store, tooling or application provider as a mechanism to query graph data.

The project aims to bring the power of graph processing and analysis to the masses by:

  • Providing a query language with full support
  • Making graph processing and analysis easier to adopt
  • Granting vendor independence to all users
  • Easing graph integration with other data platforms
